# RTV Apparel — Project Overview

## Goal
Launch a working Shopify storefront by **Friday** (3 days from scaffold).

## Tech Stack
| Layer | Technology | Notes |
|-------|------------|-------|
| Storefront | Shopify Hydrogen | Headless React/Remix framework |
| Checkout | Shopify Checkout | Standard, no customization needed |
| Fulfillment | Printful | Via Shopify app integration |
| Hosting | Oxygen (Shopify) or Vercel | TBD based on deploy simplicity |

## Definition of Done — Launch v1

### Pages Required
- [ ] Home page (hero, featured products)
- [ ] Collection page (all products grid)
- [ ] Product detail page (images, variants, add to cart)
- [ ] Cart page/drawer
- [ ] Checkout redirect (Shopify handles this)

### Policy Pages
- [ ] Shipping Policy
- [ ] Returns Policy
- [ ] Privacy Policy
- [ ] Terms of Service

### Technical Requirements
- [ ] Basic SEO (titles, descriptions, OG tags)
- [ ] Analytics placeholder (GA4 or similar ready to wire)
- [ ] Mobile responsive
- [ ] Runs without errors

### Catalog Requirements
- [ ] Printful connected to Shopify
- [ ] Minimum 7 SKUs live in Shopify (synced from Printful)
- [ ] Product images/mockups from Printful
